kingcent_stereo_demo
==========
kingcent stereo camera 

---

Dependencies
=======

- OpenCV >= 3.1

### Compile

```
mkdir build
cd build
cmake ..
make
```

### run

- Calibration

  `./binocular_calibration`

  ![](./tmp/c1.png)



- Test

  `./binocular_demo`
